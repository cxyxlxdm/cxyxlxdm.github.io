永远相信美好的事情即将发生

发表于 Nov 3 2014   |   分类于 感悟   |     |  

追求美好的东西，永远是一个人进步的源动力。

最近在看刘慈欣的《三体》，深深地被刘超凡入圣的想象力和无与伦比的格局所折服，它带给人前所未有的震撼，让人不得不深切地渴望从每一个细节去了解关于他的一切。其实细想起来，貌似也只有既是作家又是工程师的刘慈欣，才能创作出如此宏大又直达人心的作品。其汪洋自肆的思维不仅让每一位读者脑洞大开，其微妙细腻的情感又让每一位读者时时地感同身受，更加了不得的是他悲天悯人的情怀同样赋予了作品足以传世的生命力。

作品格局之大甚至大过宏观宇宙，细节之深甚至小至微观质子，想象力之丰富甚至直达11维空间，设计学科更是涵盖了几乎所有的基础前沿学科…更加难得是作品里字里行间透漏出的对宇宙的敬畏和对生命的悲悯，最让人动容。

看了《三体》，更加印证了自己之前的一些想法，语言（现在是英语）和数学作为基础学科中的基础，是一切事物甚至文明发展进步的前提，在自己擅长的领域走的越远，探得越深，就越觉着这些基础学科重要。搞IT的尤其如此，几乎所有的编程语言和技术文档都是英文，几乎所有的前沿IT技术都建立数学之上。如果再纵深下去，跨学科会显得更加重要，人文学科和理工学科的界限会更加模糊，而他们的联系会更加紧密，相互依存，又相辅相成。

创新+战略+体验+情感，创新自不多说，宏大深远的战略方针，精益求精的用户体验，再加上创作人员倾心注入的真情实感，《三体》为互联网产品提供了一个最佳的成功模板，真心希望自己今生能有机会参与到这样的产品创作中。能做到如此，确实非常困难，但是就像文章标题一样：永远相信美好的事情即将发生。虽不能至，亦可心向往之，而且生命还给了我们选择进步的机会。

谨此，与君共勉。

且行且珍惜

发表于 May 18 2014   |   分类于 感悟   |     |  

不管之前怎么想，说实话，在看到女友微信发来“没有事”的时候，我还是松了口气。我们也都太明白，如今的我们都还真的没有做好准备，无论是物质上还是心理上，都还没做好准备。虽然我也说了，如果她怀孕了，我就放弃接下来的培训计划，辞职后立马回去跟她结婚，我始终觉得，如果真出现了这种自己和女友都不愿出现的结果，对于我个人来讲，一定要承担起该负的责任。

可显然，我心里是不情愿的，最大的问题在于，我怕自己现在的处境，在做了爸爸之后，不能给自己所爱的人物质上的幸福，自己对未来做好的一些规划会因此而彻底被打乱。虽然她也告诉我，她也还真没做好做妈妈的准备，然而，我向女友表明了自己的态度，“放心，有我在”。如果她真的怀孕了，我真的会不顾一切放弃现在的一切，回去跟她结婚。这么久以来，我从来没像现在这样觉得她是那么地重要。我还开玩笑似的说，有时候我会幻想，如果自己突然有个孩子，想想也挺奇妙的。

这么久以来，我们真的是挺不容易，虽然我一直不说，但是我确信这就是我这辈子以来最值得付出的爱情。其实事后想想，真的是我们自己太紧张了，而她也一直内分泌失调，身体上有一些小毛病，导致该来的例假一直不来。这也给我们提了个醒，是时候去做个系统的身体检查，把身体上的隐患都排除掉。毕竟，身体健康才是一个人、一家人最大的福气。那些物质上的享受，在身体健康面前都是虚的。

在一块儿三年多，我的身上到处都已刻满了她的印记，无论是衣食住行，还是习惯爱好，甚至是行为思想，全都因为她一个人而改变。从一开始，谁都没想到会一起走这么远，三年时间，说起来好像不是很长一段时间，在一起七八年的多了去了。可是，我就觉得这三年已经决定了自己后多半辈子的日子。也许两个懂得珍惜的人在一起，总会把三年当作一辈子来过吧。

以后的日子，谁也说不准，且行且珍惜吧。

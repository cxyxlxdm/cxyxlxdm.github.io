与题无关

发表于 Mar 23 2011   |   分类于 感悟   |     |  

每天都是新的。明天，或是下辈子，你永远不知道哪个会先到来。不经意间，好像好多东西或是悄悄地溜走了，或是改变了原来的模样。也不知道这改变是好是坏。每个人都希望能有一个好的未来，可这也许并不是改变的原因。

前些天跟一高中非常乖又非常单纯的女孩子聊天，她告诉我：其实我真的很想叛逆一次。我突然觉得有点伤感，有过不堪经历的人，希望永远地从记忆里抹去那段曾经；记忆里一片空白的人，又觉得别人的经历即使不堪对于自己也是奢侈。其实，经历过也不一定有收获，许多时候“吃一堑，长一智”并不如老夫子讲授的那样有用。

姐订婚了，得知这个消息那一刻，觉得非常的突然，当然很快就接受了。之前她经历了一些波折，现如今也总算是有了一个结果了。有一段时间，她很不开心，总给我说她很坚强。坚强很好，可是久了，我也就怕看到“坚强”二字，因为坚强了就意味着过得不好。看到她不开心，我当然也好不到哪儿去，不过现在应该好多了吧，至少比以前好多了，过日子肯定还会有很多烦心事，别人也替不了，平平淡淡、偶尔再发发神经的生活应该很不错，追求完美往往会适得其反。我相信，只要看得开，怎样的日子都能过的有滋有味。

春天，犯困加上各种蠢蠢欲动的念头，真是最适合交配的季节啊。突然觉得21世纪最值钱的应该是媒婆了，我的若干好同学一阵骚动，都纷纷化身西厢红娘，倾巢而出，奈何错认了张生，又误识了莺莺。哪个不愿情添黄河溢，哪个不愿爱压华岳低，可谁又甘心一个将就误了终身？以我看啊，各位好青年还是收手吧，正所谓“穷则独善其身，达则兼济对象”，等待，终究会在正确的时间，等到正确的人的。

我还得感谢英明神武的教务处领导们，感谢他们给我安排了一个灰常给力的课表，让我有时间起早贪黑去做钟点工，能有这样一份经历，我很开心也很感激，而且这个学期一直到现在也还没遇到什么难处。接触更多的人，遭遇更多的事，承担更大的责任，付出更多的努力，我不奢望有更好的结果，只求能以一颗如履薄冰的心态让自己得到一些认可。谢谢和我一起的苏阿姨，多好的人啊，做了两周的钟点工，通过苏阿姨，我也了解到学校里好多老师、教授也都是通过自己的努力一步步走到现在，他们的故事分明就是一幕幕励志短剧。每个人的成功总要经历这样那样的磨难，任何时候，人都不该也不能放弃自己的。

怀着感恩的心去生活不该只是靠自己时不时的提醒，而应该成为习惯。面包会有的，牛奶也会有的，少年，好好干吧。

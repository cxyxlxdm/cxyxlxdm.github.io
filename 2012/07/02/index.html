Tomcat开机启动脚本

发表于 Jul 2 2012   |   分类于 Linux   |     |  

在/etc/init.d下创建脚本文件tomcat，然后赋予其执行权限：chmod +x tomcat.

使用命令service tomcat start/restart/stop来启动/重启/停止服务。
